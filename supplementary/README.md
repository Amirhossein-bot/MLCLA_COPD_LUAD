# Supplementary Materials

This directory contains the supplementary materials for the MLCLA-Net COPD-LUAD manuscript.

## Files

- `Supplementary_Tables.docx`: final supplementary tables document.
- `Supplementary_Figures/`: directory for supplementary figures cited in the manuscript.
- `Supplementary_Archive_S1/`: directory for raw enrichment outputs and extended reproducibility files.

## Note

The supplementary tables are provided as a DOCX file because this is the format of the final uploaded supplementary table document. If the target journal requests spreadsheet format, convert the final accepted table version to XLSX before submission.
