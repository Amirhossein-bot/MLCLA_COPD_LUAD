# Results

This folder stores selected processed outputs supporting the MLCLA-Net manuscript.

Raw TCGA and GEO files should not be stored here. Instead, this folder should contain processed outputs generated by the repository scripts.

## Subfolders

- `selected_mlcla_outputs/`: selected MLCLA-Net community assignments, retained communities, convergence history, and shared identifier outputs
- `baseline_comparison/`: Louvain, Leiden, Label Propagation, Greedy Modularity, connected-components comparison outputs
- `enrichment/`: MLCLA and baseline enrichment outputs
- `external_validation/`: independent GEO candidate-level, directional-concordance, module-level, and pathway-validation outputs
- `survival/`: TCGA-LUAD survival-support outputs
- `figures/`: manuscript figure source files and final exported figures

## Recommended practice

Use clear filenames that match the manuscript and supplementary materials. For example:

```text
results/selected_mlcla_outputs/MLCLA_selected_run_summary.xlsx
results/baseline_comparison/MLCLA_baseline_method_comparison.xlsx
results/enrichment/BIB_supplementary_enrichment_tables.xlsx
results/external_validation/external_validation_summary.xlsx
results/survival/TCGA_LUAD_survival_support_outputs.xlsx
results/figures/Figure5_MLCLA_convergence_profile.png
```
